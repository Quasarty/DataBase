# Вариант 13. Техническое обслуживание станков
1) Получить классификацию видов ремонта по заданным диапазонам цен. Диапазоны цен задаются в отдельной таблице. Совокупность видов ремонта представить в виде массива строк.
2) Получить прибыль предприятия за каждый месяц заданного года.
3) Получить список из 3 самых «ненадежных» (часто ремонтируемых) станков. Для каждого их них указать дату самого первого и самого последнего ремонта.
4) Получить суммарная продолжительность ремонта всех станков. Выявить, продолжительность ремонта каких станков больше средней продолжительности ремонта всех станков.
5) Получить кросс-таблицу из итоговой таблицы R(«Станок», «год», «количество ремонтов»). Значения атрибута «год» представить в виде колонок кросс-таблицы.
